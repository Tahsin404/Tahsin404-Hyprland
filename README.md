## Installation

1. Clone the repo:
   ```bash
   git clone [https://github.com/Tahsin404/Tahsin404-Hyprland.git](https://github.com/Tahsin404/Tahsin-Hyprland.git)
   cd Tahsin-Hyprland

2. Make install.sh executable and run
   ```bash
   chmod +x install.sh
   ./install.sh
